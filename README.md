# maung-backend
Backend Module of Maung 2019 site

## Endpoints

* FindAllMembers (/members)
Arguments :
    - page (int)
    - limit (int)

* SearchMembers (/search)
Arguments :
    - page (int)
    - limit (int)
    - keyword (string)

## Run on Local
* Clone this repo
* Go to `maung_backend` directory
* Run `python3 manage.py runserver`
